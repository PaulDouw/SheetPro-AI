Formulas & Commands

SheetPro AI provides a variety of functions to call different AI models and output formats. Simply install the add-on, configure your API keys in Google Sheets, then use these formulas.

1. Core Functions

Formula Description Example
AI(prompt) Calls your configured provider and model (set in sidebar) with a given prompt. =AI("Explain quantum physics in 3 bullet points.")
AI_LIST(prompt) Returns each line of the AI response in separate rows (vertical list). =AI_LIST("Give me 5 marketing tips.")
AI_TABLE(prompt) Tries to parse the AI response into a 2D table in Google Sheets. =AI_TABLE("Provide a table of the top 5 NFL teams and their stadiums.")

2. Model-Specific Formulas

These override the default provider/model set in the sidebar. You still need to provide the corresponding API key.

Formula Provider / Model Example Usage
GPT_4O(prompt) OpenAI — GPT-4o =GPT_4O("Summarize this article:" & A1)
GPT_4O_MINI(prompt) OpenAI — GPT-4o-mini =GPT_4O_MINI("Draft a quick email reply.")
GEMINI_2_0_FLASH(prompt) Google Gemini 2.0 Flash =GEMINI_2_0_FLASH("Explain the difference between cats and dogs.")
GEMINI_2_0_FLASH_LITE(prompt) Google Gemini Flash Lite =GEMINI_2_0_FLASH_LITE("Generate ideas for a blog post.")
GEMINI_1_5_PRO(prompt) Google Gemini 1.5 Pro =GEMINI_1_5_PRO("Advanced business plan strategies.")
DEEPSEEK_CHAT(prompt) DeepSeek — Chat =DEEPSEEK_CHAT("Explain blockchain in simple terms.")
DEEPSEEK_REASONER(prompt) DeepSeek — Reasoner =DEEPSEEK_REASONER("Perform advanced logical reasoning.")
SEARCH_WEB(prompt) Perplexity AI — Web Search =SEARCH_WEB("Who won the 2023 US Open Tennis?")

3. Web & Scraping Functions

Formula Description Example
SCRAPE(url) Fetches raw text from a webpage (removing HTML, scripts, etc.). =SCRAPE("https://example.com/page")
ANALYZE_URL(url, instruction) Combines SCRAPE + AI to analyze or summarize a webpage. =ANALYZE_URL("https://example.com/article", "Summarize the key points.")
