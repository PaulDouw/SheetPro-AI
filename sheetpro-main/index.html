Fonctionnalités Clés

Configuration & Paramétrage

Choisissez votre fournisseur IA (OpenAI, Gemini, DeepSeek) et configurez vos clés API dans une barre latérale intuitive. Vos réglages sont sauvegardés automatiquement.

Fonctions AI Génériques

Utilisez AI(), AI_LIST() ou AI_TABLE() pour envoyer vos prompts et obtenir des réponses sous forme de texte, listes ou tableaux directement dans Sheets.

Fonctions Spécifiques & Web

Déclenchez directement des modèles spécifiques (GPT_4O, GEMINI_2_0_FLASH, etc.) ou récupérez du contenu Web via SCRAPE() et ANALYZE_URL().

Vos Avantages

Multi-fournisseurs, formules adaptées, scraping et analyse avancée… Tout cela gratuitement. Vous payez uniquement les frais d’utilisation facturés par votre fournisseur IA.

Les Formules & Commandes

SheetPro AI met à disposition un ensemble complet de formules pour exploiter pleinement l’IA dans vos Google Sheets.

Fonctions de Base

Formule Description Exemple
AI(prompt) Utilise le fournisseur et le modèle configurés pour traiter votre prompt. =AI("Expliquer la physique quantique en 3 points")
AI_LIST(prompt) Retourne la réponse sous forme de liste verticale (une ligne par élément). =AI_LIST("5 conseils marketing")
AI_TABLE(prompt) Analyse le texte pour générer un tableau (2D array) dans Sheets. =AI_TABLE("Tableau des 5 villes les plus peuplées")

Fonctions Avancées

Formule Modèle / Fournisseur Exemple d'Usage
GPT_4O(prompt) OpenAI – GPT-4o =GPT_4O("Résumé de l'article :" & A1)
GEMINI_2_0_FLASH(prompt) Google Gemini 2.0 Flash =GEMINI_2_0_FLASH("Différences entre chats et chiens")
DEEPSEEK_CHAT(prompt) DeepSeek – Chat =DEEPSEEK_CHAT("Expliquer la blockchain")
SEARCH_WEB(prompt) Perplexity – Web Search =SEARCH_WEB("Meilleur laptop 2023")

Fonctions de Scraping

Formule Description Exemple
SCRAPE(url) Extrait le texte d'une page Web en supprimant le HTML et scripts. =SCRAPE("https://exemple.com/page")
ANALYZE_URL(url, instruction) Combine SCRAPE + AI pour résumer ou analyser une page Web. =ANALYZE_URL("https://exemple.com/article", "Résumé des points clés")

Politique de Confidentialité

Dernière mise à jour : 1er janvier 2025

Chez SheetPro AI, nous nous engageons à protéger votre vie privée. Cette politique explique comment nous traitons vos informations lorsque vous utilisez notre add-on Google Sheets.

1. Collecte des données
Nous ne collectons ni ne stockons vos données de feuille de calcul. Les requêtes sont traitées en temps réel.

2. Utilisation de vos informations
Vos prompts sont transmis en toute sécurité à nos partenaires IA (OpenAI, Google Gemini, DeepSeek, etc.) pour générer des réponses. Nous ne conservons pas ces données.

3. Sécurité
Les données sont transmises via des connexions sécurisées (HTTPS/TLS). Nous prenons toutes les mesures nécessaires pour protéger vos informations.

4. Fournisseurs tiers
Nos partenaires peuvent avoir leurs propres politiques de confidentialité. Nous vous invitons à les consulter.

5. Modifications
Cette politique peut être mise à jour. La date de dernière mise à jour sera indiquée ci-dessus.

6. Contact
Pour toute question, contactez-nous à privacy@sheetpro.com.
