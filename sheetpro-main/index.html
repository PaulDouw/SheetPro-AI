Formulas & Commands

SheetPro AI provides a comprehensive set of formulas to harness the power of AI directly in your Google Sheets.

Core Functions

Formula Description Example
AI(prompt) Uses your configured provider and model to process your prompt. =AI("Explain quantum physics in 3 bullet points")
AI_LIST(prompt) Returns the response as a vertical list (one item per row). =AI_LIST("Give me 5 marketing tips")
AI_TABLE(prompt) Attempts to parse the response into a 2D table within Sheets. =AI_TABLE("List the top 5 most populous cities")

Advanced Functions

Formula Provider / Model Example Usage
GPT_4O(prompt) OpenAI – GPT-4o =GPT_4O("Summarize this article:" & A1)
GEMINI_2_0_FLASH(prompt) Google Gemini 2.0 Flash =GEMINI_2_0_FLASH("Describe the differences between cats and dogs")
DEEPSEEK_CHAT(prompt) DeepSeek – Chat =DEEPSEEK_CHAT("Explain blockchain in simple terms")
SEARCH_WEB(prompt) Perplexity – Web Search =SEARCH_WEB("Who won the 2023 US Open Tennis?")

Scraping Functions

Formula Description Example
SCRAPE(url) Extracts text from a webpage by removing HTML and scripts. =SCRAPE("https://example.com/page")
ANALYZE_URL(url, instruction) Combines SCRAPE and AI to summarize or analyze a webpage. =ANALYZE_URL("https://example.com/article", "Summarize the key points")
